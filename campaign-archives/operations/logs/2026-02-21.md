# 2026-02-21 运行日志 (系统加固与本地化失败)

## 事件总结
1. **系统更新导致鉴权丢失**：OpenClaw 从 2026.2.17 更新到 2026.2.21-2 时，Agent 目录被重置，导致 API Key (auth-profiles.json) 丢失。
2. **本地模型死锁**：尝试在 Mac mini 上运行 13GB 的 MiniMax-m2.5。由于 context 堆积到 115k (超出 40k 限制 288%)，且显存不足，导致 Ollama 频繁出现僵尸进程 (SIGKILL 137) 并锁死端口 11434。
3. **系统响应异常**：由于模型处理时间过长，触发了网关重试或重复回复。

## 加固措施 (Reliability 2.1)
- **Watchdog 2.0**: 部署在 `~/.openclaw/watchdog.sh`，每分钟通过 crontab 运行，负责回填 API Key 和 重启 Ollama。
- **配置持久化**: 备份 auth 文件到 `~/.openclaw/config_safe/`。
- **系统服务化**: 尝试注册 launchd 服务，但受限于 sudo 权限。

## 待办与教训
- **13GB MiniMax 在 115k context 下不可用**：必须限制 context 或增加超时容忍。
- **禁止并发调用本地模型**：防止 GPU 锁死。
- **清理僵尸进程**：必须使用 `pkill -9` 且清除 `.lock` 文件。
